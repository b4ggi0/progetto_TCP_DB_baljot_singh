# Progetto Server e Client

Questo progetto consiste in un'applicazione server e client scritta in Python per gestire interazioni con un database. 
Il server fornisce API per eseguire operazioni di creazione, lettura, modifica ed eliminazione sui dati del database,
mentre il client interagisce con il server per eseguire queste operazioni.

## Requisiti

- Python 3.x
- Librerie Python necessarie: `socket`, `json`, `mysql-connector-python`


## Configurazione


##Installa le dipendenze eseguendo il comando:
   ```bash
   pip install mysql-connector-python



##Il file progetto_server.py implementa il lato server dell'applicazione. Avvia il server eseguendo il comando:
python progetto_server.py
Il server sarà in ascolto su un indirizzo IP e una porta specificati nel codice.


##Il file progetto_client.py implementa il lato client dell'applicazione. Esegui il client con il comando:
python progetto_client.py
Il client si connetterà al server e fornirà un'interfaccia per interagire con il database.


##Utilizzo
Segui le istruzioni fornite dall programma per eseguire operazioni di creazione, lettura, modifica ed eliminazione sui dati del database.
